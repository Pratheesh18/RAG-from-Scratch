# RAG System Using LangChain

This project implements a simple **Retrieval-Augmented Generation (RAG)** system using LangChain.

The system retrieves relevant document chunks from a vector database and generates answers using an LLM grounded in the retrieved context.

---

## Overview

This project demonstrates:

- Document loading and processing
- Text chunking
- Embedding generation
- Vector similarity search
- Context-aware answer generation using an LLM

The goal is to understand how RAG works internally using LangChain components.
